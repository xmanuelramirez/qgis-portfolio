## What's inside

This is a single-page interactive portfolio showcasing selected QGIS and geospatial analysis projects. It is built as a self-contained HTML file — no backend, no build step, just open it in any browser or visit the live URL above.

### Projects featured

| # | Project | Location | Key techniques |
|---|---------|----------|----------------|
| 01 | **Hydraulic Sectorization — Water Tank Pressure Zones** | Ciudad Juárez, Chihuahua | DTM raster analysis, gravitational pressure head, AutoCAD integration, PyQGIS |
| 02 | **Infrastructure Sectorization — Municipal Hydraulic Planning** | León, Guanajuato | QGIS Atlas, multi-network analysis (potable + treated water + stormwater + sanitary sewer), dynamic map series |
| 03 | **Hydraulic Capacity & Risk — Urban Stream Assessment** | León, Guanajuato | 0.5 m LiDAR DTM, 25-yr design storm, cross-section profiling, traffic-light risk classification |

## Running locally

No installation needed — just open `index.html` in any modern browser:

```bash
# Clone the repo
git clone https://github.com/xmanuelramirez/qgis-portfolio.git

# Open in browser (macOS)
open qgis-portfolio/index.html

# Open in browser (Windows)
start qgis-portfolio/index.html
```
